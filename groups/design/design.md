# 设计创意交流群

本页面收录了各类设计创意相关的微信群，涵盖UI/UX设计、平面设计、插画等领域。

## 贡献指南

如果您有新的设计创意相关微信群可以添加，请按照以下格式提交：
1. Fork 本仓库
2. 在此文件中添加群组信息
3. 通过 Issues 方式提交

### 添加格式
```
### 群组名称
**行业/领域**: [填写具体领域]
**群主/推荐人**: [可选]
**入群方式**: 
- 二维码: [在此处添加二维码图片]
- 邀请人: [联系人微信号]
**群介绍**: [简短描述群组特色和讨论话题]
**活跃度**: [高/中/低]
```

## 注意事项

- 请遵守群规，尊重群内成员
- 不发布广告或不当内容
- 积极参与有价值的话题讨论
